# AKI NO MASS

**Sistema de Gestión Comercial Multicanal (DEMO funcional offline).**

AKI NO MASS es una demo de alta fidelidad orientada a un emprendimiento de venta de ropa con captación en TikTok, WhatsApp, Instagram, Facebook y Telegram.

## ¿Qué incluye esta demo?

- Login visual profesional (sin autenticación real).
- Panel administrativo SPA en una sola página.
- Módulos: Dashboard, Clientes, Productos, Pedidos, Canales, Seguimiento, Reportes y Configuración.
- Datos demo precargados y persistencia con `localStorage`.
- Operaciones CRUD locales (crear, editar, eliminar), búsquedas y filtros.
- Reportes visuales con barras usando solo HTML, CSS y JavaScript puro.

## Importante

- Esta es una **demo funcional**, no un backend productivo.
- No usa Laravel, backend, PHP, base de datos real, React, Vue ni Node.
- No requiere instalación ni paquetes externos.

## Cómo abrirlo

1. Descarga o clona este repositorio.
2. Abre el archivo `index.html` con doble clic en cualquier navegador moderno.
3. Ingresa con cualquier usuario/contraseña en el login visual.

✅ No necesitas servidor local: funciona directamente desde el sistema de archivos.
